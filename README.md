# Crusher Design Assistant

A lightweight engineering tool for roll crusher preliminary design and calculation.

## Functions
- Roller linear speed calculation
- Capacity estimation (empirical model)
- Motor power estimation

## Usage
Run the Python script:
python calculator.py
Input parameters:
- Roller diameter (mm)
- Speed (rpm)
- Width (mm)

## Example
Input:
- Diameter: 360 mm
- Speed: 300 rpm
- Width: 2100 mm

Output:
- Speed: 5.65 m/s
- Capacity: ~xxx t/h
- Power: ~xxx kW

## Background
This tool is used in daily engineering work for quick calculation and preliminary design of crushing equipment.
